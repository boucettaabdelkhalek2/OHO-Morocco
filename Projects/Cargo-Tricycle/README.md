# 🚲 Cargo Tricycle – Open Hardware Project (OHO Morocco)

## 🌍 Overview
This project is an open-source **cargo tricycle (front loader)** designed for transporting up to **150kg** using a welded steel frame.

It is based on the Werkstatt Lastenrad workshop design and adapted for OHO Morocco.

---

## ⚙️ Key Features
- Load capacity: 150kg
- Steel welded frame
- 3x 26" wheels
- Front cargo box (0.9m × 0.8m × 0.6m)
- Dual front braking system
- Low gear ratio recommended

---
