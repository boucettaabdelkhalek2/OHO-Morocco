# Motor System

## ⚙️ Motor Type
Option 1: Hub Motor (recommended)
- Easy installation
- Low maintenance

Option 2: Mid-drive motor
- Higher torque
- Better performance uphill

## ⚡ Power
- 250W (legal / low power)
- 500W – 1000W (recommended for cargo)

## 🧮 Torque Estimation
- Required torque increases with load
- Target: high torque for 100kg load

## 🔋 Efficiency
- Optimize controller current
- Avoid overheating
- Use proper gearing (if mid-drive)
