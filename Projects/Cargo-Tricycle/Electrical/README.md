# Electrical System

## 🔋 Battery
- Type: Li-ion 18650
- Voltage: 48V (13S configuration)
- Capacity: 10Ah – 20Ah

## 🧠 BMS
- 13S BMS protection
- Overcharge / overdischarge protection
- Temperature monitoring (optional)

## ⚙️ Controller
- 48V controller
- Current: 20A – 30A
- Compatible with hub motor

## 🔌 Wiring
- Battery → BMS → Controller → Motor
- Throttle connected to controller
- Fuse between battery and controller

## 🔒 Safety
- Fuse (important)
- Proper insulation
- Heat management for MOSFET
