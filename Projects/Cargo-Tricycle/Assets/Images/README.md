# 🚲 Cargo Tricycle (Open Hardware Project)

![Preview](Assets/Images/Render/200px-Tricycle_cargo_bike.png)

## 🌍 Overview
Open-source cargo tricycle designed for heavy transport (up to **150kg**).

- 🔧 DIY Steel Frame
- ⚡ Sustainable Mobility
- 🧰 Fully Documented (CAD + BOM + Assembly)
- 📦 Front Cargo Box

## 🏗️ Assembly Guide

1. Frame welding (base structure)
2. Front cargo box assembly
3. Steering system integration
4. Wheel mounting (3x)
5. Brake system (dual front brakes)
6. Final testing
